# ゲームルール詳細

## 1. ゲーム概要

LLM人狼は、1人の人間プレイヤーが8体のAIと人狼ゲームを行うWebアプリケーションです。

| 項目 | 内容 |
|------|------|
| プレイヤー数 | 9人（人間1 + AI8） |
| 役職構成 | 人狼2、狩人1、占い師1、霊媒師1、狂人1、村人3 |
| 配役方法 | ランダム、または人間プレイヤーが役職を選択 |

## 2. 役職一覧と能力

| 役職 | 陣営 | 能力 |
|------|------|------|
| 村人 | 村人陣営 | 特殊能力なし。議論と投票で人狼を見つける |
| 占い師 | 村人陣営 | 毎晩1人を占い「人狼か否か」を知る |
| 霊媒師 | 村人陣営 | 処刑された人が人狼だったか否かを知る |
| 狩人 | 村人陣営 | 毎晩1人を護衛し、人狼の襲撃から守る |
| 狂人 | 人狼陣営 | 特殊能力なし。占いでは村人と判定される |
| 人狼 | 人狼陣営 | 毎晩1人を襲撃して殺害する。人狼同士は互いを認識している |

### 占い師の制約

- 自分自身は占えない
- 同じ人を複数回占えない
- 夜に襲撃されたら、その夜の占い結果は得られない

### 霊媒師の制約

- 処刑があった翌朝に結果を得る
- 夜に襲撃されたら、その夜の霊媒結果は得られない

### 狩人の制約

- 自分自身は護衛できない

### 狂人の制約

- 特殊能力なし
- 占い結果は「村人」と判定される
- 勝利条件は人狼陣営と同じ

### 人狼の制約

- 自分自身は襲撃できない
- 他の人狼は襲撃できない
- 襲撃対象は自分・人狼以外の生存者
- 人狼が複数生存している場合、代表の1人が襲撃対象を決定する

## 3. ゲームの流れ

ゲームは「昼フェーズ」と「夜フェーズ」を繰り返して進行します。

```
ゲーム開始（役職配布・番号決定）
  │
  ├─ Day 1（昼）: 議論（1巡） → 投票 → 処刑 → 勝利判定
  │
  ├─ Night 1（夜）: 占い / 護衛 / 襲撃 → 勝利判定
  │
  ├─ Day 2（昼）: 占い結果・霊媒結果通知 → 議論（2巡） → 投票 → 処刑 → 勝利判定
  │
  ├─ Night 2（夜）: 占い / 護衛 / 襲撃 → 勝利判定
  │
  └─ ... 勝利条件を満たすまで繰り返し
```

### 初日の扱い

- Day 1 から通常通り議論・投票を行う（スキップしない）

### 発言ルール

| 項目 | 内容 |
|------|------|
| 発言巡数 | 1日目は1巡、2日目以降は2巡 |
| 発言順（初日） | ゲーム開始時にランダムで番号を決定 |
| 発言順（2日目以降） | 襲撃された人の次の番号から開始 |
| 発言順の維持 | 同じ日の複数ラウンドでは発言順は変わらない |
| 発言順の変更条件 | 発言順が変更されるのは夜の襲撃後のみ（処刑では変更されない） |

### 占い結果の通知

- 占い結果は翌朝の議論開始前に占い師本人へ通知される

### 霊媒結果の通知

- 霊媒結果は翌朝の議論開始前に霊媒師本人へ通知される

## 4. 投票ルール

| 項目 | 内容 |
|------|------|
| 投票方式 | 公開投票（誰が誰に投票したか全員に見える） |
| 自己投票 | 不可 |
| 棄権 | 不可（必ず誰かに投票する） |
| 同数時 | ランダム処刑（同票の候補者から抽選） |
| 処刑時の役職公開 | 公開しない |

## 5. 夜フェーズルール

夜フェーズでは、占い → 護衛 → 襲撃 の順で行動を解決する。

### 占い（占い師）

- 占い師は生存者の中から1人を選んで占う
- 占い結果（人狼か否か）は翌朝通知される
- 対象の制約は [占い師の制約](#占い師の制約) を参照

### 護衛（狩人）

- 狩人は生存者の中から1人を選んで護衛する
- 護衛対象が人狼の襲撃対象と一致した場合、襲撃は阻止される（護衛成功）
- 対象の制約は [狩人の制約](#狩人の制約) を参照

### 襲撃（人狼）

- 人狼は生存者の中から1人を選んで襲撃する
- 人狼が複数生存している場合、代表の人狼が襲撃対象を決定する
- 対象の制約は [人狼の制約](#人狼の制約) を参照

### 霊媒（霊媒師）

- 昼フェーズで処刑が行われた場合、霊媒師は処刑された人が人狼だったか否かを知る
- 結果は翌朝通知される

## 6. 勝利条件

| 陣営 | 条件 |
|------|------|
| 村人陣営 | 人狼を全滅させれば勝利 |
| 人狼陣営 | 人狼以外の生存者が人狼の数以下になれば勝利（狂人は人狼ではないためカウントに含む） |

### 判定タイミング

- 昼フェーズの処刑後
- 夜フェーズの襲撃後

各フェーズ終了時に勝利条件を判定し、条件を満たしていればゲーム終了となる。
